# Slack2Discord App

A Slack application that will forward messages from any Slack workspace to a Discord channel of your choosing!
Utilizes Discord webhooks and the Slack Events API.

### Built with

- <a href="https://github.com/slackapi/python-slack-events-api" target="_blank">Slack Events API Adapter for Python</a>
- MongoDB Cloud
- Flask
- Discord Webhooks

Hosted on <a href="https://heroku.com/" target="_blank">Heroku</a>
